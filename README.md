# 🔐 Hash Generator

A simple Python project to generate cryptographic hashes from user input.

## 🚀 Features

- MD5 Hash
- SHA1 Hash
- SHA256 Hash
- Keccak-256 Hash

## 🛠️ Built With

- Python
- hashlib

## ▶️ Usage

```bash
python main.py
```

- Hash Algorithms (MD5, SHA1, SHA256, Keccak-256)
